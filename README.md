# 📊 India Sales Intelligence Dashboard

An advanced, interactive sales analytics platform built with **Streamlit** and **Plotly**. This dashboard provides a 360-degree view of sales performance, from executive summaries to predictive machine learning insights.

## 🚀 Features
- **Executive Home:** Real-time KPIs (Revenue, Deals, Talk Time, Conversion).
- **Descriptive Analytics:** Deep dive into rep-level performance and trends.
- **Diagnostic Tools:** Pareto analysis, outlier detection, and performance heatmaps.
- **Perspective Benchmarks:** Radar charts and cohort analysis for team comparison.
- **Predictive & Prescriptive:** Machine learning-driven revenue forecasting and "What-If" scenario simulations.

## 🛠 Tech Stack
- **Frontend:** Streamlit
- **Visualization:** Plotly (Express & Graph Objects)
- **Data Processing:** Pandas, NumPy
- **Machine Learning:** Scikit-Learn (Decision Tree Regressor)
